Bitcoin aman ditangan anda dengan dompet open source, multisignature dari BitPay. 
Pengguna Copay dapat menyimpan dana secara individu atau finansial bersama dengan pengguna lain dengan dompet multisignature, yang mencegah pembayaran yang tidak sah dengan mengharuskan persetujuan bertingkat. Berikut adalah beberapa cara Copay dapat digunakan dengan orang lain:

Simpanan untuk liburan atau pembelian bersama dengan teman-teman 
Melacak pengeluaran dan tunjangan keluarga
Untuk mengelola dana dan pengeluaran bisnis, klub, atau organisasi

Kami membangun fitur-fitur berikut ke versi Copay untuk dompet bitcoin yang tidak berkompromi mengenai keamanan atau aksesibilitas:

Pembuatan banyak dompet dan manajemen didalam aplikasi 
Keamanan Intuitif multisignature untuk dompet pribadi atau bersama
Kemudahan aliran proposal pengeluaran untuk dompet bersama dan pembayaran berkelompok
Hierarchical deterministic (HD) address generation and wallet backups 
Device-based security: all private keys are stored locally, not in the cloud
Support for Bitcoin testnet wallets
Synchronous access across all major mobile and desktop platforms
Payment protocol (BIP70-BIP73) support: easily-identifiable payment requests and verifiably secure bitcoin payments
Support for 150+ currency pricing options and unit denomination in BTC or bits
Email notifications for payments and transfers
Customizable wallet naming and background colors
10 supported languages (EN, JP, FR, ES, IT, DE, PT, RU, EL, TR)

Copay is free and open source software run on non-proprietary servers, so there's no need to rely on any company for continuous support. Anyone can review or contribute to Copay's source code on GitHub (https://github.com/bitpay/copay).
